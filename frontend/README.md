# Moyano Motos - Frontend

Panel de gestion construido con React 19 y Vite.

## Modulos

- Autenticacion y rutas protegidas.
- Dashboard de ventas y cobranzas.
- Clientes.
- Motos.
- Ventas y planes de cuotas.
- Cuotas y pagos.
- Recibos imprimibles.
- Reportes.
- Usuarios y roles.

## Configuracion

La API se consulta en `http://localhost:8000/api` por defecto. Para usar otra URL:

```env
VITE_API_URL=https://api.ejemplo.com/api
```

## Comandos

```bash
npm install
npm run dev
npm run lint
npm run build
```
